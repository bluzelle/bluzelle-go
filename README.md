## Getting Started

- Get package
```bash
go get github.com/wlwanpan/bluzelle-go
```

- Import
```go
import "github.com/wlwanpan/bluzelle-go"
```

- Initialize
```go
blz := Connect("127.0.0.1", 51010, "80174b53-2dda-49f1-9d6a-6a780d4")
```
